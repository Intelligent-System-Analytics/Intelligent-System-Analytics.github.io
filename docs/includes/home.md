# Introduction to IoT Edge

Azure IoT Edge is a device-focused runtime that enables you to deploy, run, and monitor containerized Linux workloads.

Example: Smart Camera, Temperature Sensor

###### Required software
- VS Code
- Postman
- Putty
- Docker
